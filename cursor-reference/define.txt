Pointer: 
>> In css, a pointer usually refers to the mouse cursor style that appears when a user moves the mouse over an element.
>> Is a value of the cursor property used to change the mouse cursor to a hand icon, indicating that the element is clickable.

Syntax:
>> selector{ cursor: pointer; }

Example:
html: <button class="btn">Click Me</button>
css: .btn{ cursor: pointer; }

>> When the user moves the mouse over the button, the cursor changes to a hand pointer, showing that it can be clicked.

Used for?
>> Improves user Experience
>> Shows that an element is interactive
>> Commonly used for buttons, links and clickable cards
